A DEFENSE FOR lon AS YES (to anu seme)

Many tokiponists say you can't use lon to answer 'anu seme' questions for many different reasons. I will answer all the criticisms I have seen.

1. Not all languages have an all-purpose "yes", so this is Englishy.

It's true that not all languages have an all-purpose "yes". But this is only a "yes" for the "anu seme" type of question, not for all questions. This type of question can be answered with "lon", while "sina pona ala pona?" can be answered with "pona".

2. It doesn't follow lipu pu.

It's perfectly fine to follow lipu pu! However, some common grammar don't follow lipu pu, such as using alasa as a preverb, kin and namako with unique meanings, and using nimi ku suli. This is common enough to be one of those.

3. "Correct" or "true" doesn't work as "yes".

At first, it makes sense. Responding to "Did you eat a good fish, or what?" with "Correct." doesn't exactly work. However, there is a problem, and that involves how "sina moku e kala pona, anu seme?" is translated. In the toki pona sentence, "anu seme" comes after a statement. But in the English sentence, "or what" comes after a question. So a better translation would be "You ate a good fish, right?". Now saying "Correct." actually makes sense, which justifies using "lon" to answer these types of questions.
