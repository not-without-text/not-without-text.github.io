A DEFENSE FOR lon AS YES (to anu seme)

Many tokiponists say you can't use lon to answer 'anu seme' questions for many different reasons. I will answer all the criticisms I have seen.

1. Not all languages have an all-purpose "yes", so this is Englishy.

It's true that not all languages have an all-purpose "yes". But this is only a "yes" for the "anu seme" type of question, not for all questions. This type of question can be answered with "lon", while "sina pona ala pona?" can be answered with "pona".
